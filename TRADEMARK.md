## Permitted Use

You **may** use the Whisq name and logo:

- To refer to the Whisq framework in articles, blog posts, tutorials, and talks
- To indicate that your project is built with Whisq ("Built with Whisq", "Powered by Whisq")
- In community resources, educational materials, and open-source projects that integrate with Whisq
- As a dependency name in package.json, import statements, and documentation

## Restrictions

You **may not**:

- Use the Whisq name or logo in a way that suggests official endorsement or affiliation with Markot s.r.o. without written permission
- Use the Whisq name as part of your own product name, company name, or domain name (e.g., "WhisqCloud", "whisq-hosting.com")
- Modify, distort, or alter the Whisq logo or mascot
- Use the Whisq logo as your own app icon, favicon, or brand mark
- Sell merchandise using the Whisq name or logo without permission

## Community Projects

Community projects that extend, integrate with, or complement Whisq are encouraged. Use names like:

- "whisq-plugin-xxx" (for plugins/extensions)
- "awesome-whisq" (for resource lists)
- "xxx-for-whisq" (for integrations)

These naming patterns are permitted and don't require approval.
